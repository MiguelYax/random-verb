# For developers

## Clone the repository

```
git clone git@github.com:MiguelYax/random-verb.git
```

## Install dependencies 
```
mpn install
```

## Build a distribution files
```
npm run build
```

## Check code style
```
npm run lint
```

## Live server with parcel

```
npm run dev
```

## Run unit tests
```
npm run test
```

# Run coverage 
```
npm run coverage
```
